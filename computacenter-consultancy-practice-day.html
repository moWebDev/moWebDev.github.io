Consultancy Practice Day

At the Computacenter, they have held an event for consultants, hence the name Consultancy Practice Day. This project initially was supposed to be an app but as the deadline grew closer, there wasn’t enough time for them to develop an app. Instead, this project was passed on to me to develop an “app like” microsite. I had free reign in design and development of this site which was something I wanted to do and had the pleasure of completing. This microsite was developed using HTML, CSS and JQuery.
