To-do App

This project was to build a to-do app where the user can add new to-do items, check an item when completed and delete an item from the list. This app was built using pure JavaScript with the use of HTML and CSS. The purpose of building this app was to practice my current skills in JavaScript as I am still a beginner.

The above images demonstrate how the app functions. At the current moment, the data is stored within the browser. This means when the user refreshes the page, all data is lost. This app will work far better when the data is stored elsewhere such as a server or the memory of the device.
