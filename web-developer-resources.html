Tools

The following tools are highly recommended. More tools will be added in the future

Atom Code Editor

This is my favourite code editor to use. It is highly customisable allowing you to feel comfortable when you are coding. This editor can be linked directly with Github repository as this editor was developed by them.

Get Atom

Sublime Text Editor

This editor is excellent for coding. More features have been added in the latest version allowing it to be more easier to use. This is the text editor I chose when I ditched Adobe Dreamweaver.

Get Sublime Text

Brackets Editor

This is another brilliant code editor which I used before moving onto Atom. This editor has live preview functionality. As you code, you can see the changes appear in the browser. I would recommend this to people who are new to web development. It does have few other features that will help a newbie.

Get Brackets

Adobe Photoshop

This application is one of the best photo editing software out there in the market. It includes: image manipulation, typography, 3D modelling and many other functionalities. Although there are many similar software out there today, this is the most recommended as the industry standard. If you are a Web designer/developer creating user interfaces, this is for you.

Get Photoshop

Adobe Illustrator

This is another recommended Adobe application which is used for designing and creating new icons, logos and various other vector based graphics. It would be useful to have this software when editing certain logos.

Get Illustrator

Books

My book list is not limited to those below. I featured these books because I have read them and have found them helpful.

HTML & CSS by Jon Duckett

is an excellent book to start learning about Web Development. It is well written and contains colourful graphics that will aid your understanding. I highly recommend you to read this book alongside the practical Codecademy course on HTML and CSS. This will speed up your learning process and understanding.

Get The Book Now

JavaScript & JQuery by Jon Duckett

Like the previous book by Jon Duckett, he uses similar style of explaining about JavaScript & JQuery. This is the current book that I am reading on JavaScript. It also features various JavaScript tasks which you could practice. There are further available resources from their website for practicing. At this current moment in time, JavaScript is very popular and that will continue to increase as many capabilities are being added such as developing desktop applications etc.

Get The Book Now

JavaScript for Kids: A Playful Introduction to Programming

I know what you are thinking but check it out, you will understand why this is on the list. If you are new to programming and have never studied or practice programming. Then this book is for you as it will help you understand the concepts and practices of programming in an easier way. JavaScript is scripting language that features programming elements/concepts.

Get The Book Now
